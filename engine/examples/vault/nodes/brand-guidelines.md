# Acme Corp Brand Guidelines

Our brand represents who we are and how we communicate with the world.

## Brand Values

1. **Innovation**: We push boundaries
2. **Trust**: We deliver on promises
3. **Simplicity**: We make complex things easy
4. **Impact**: We create meaningful change

## Logo Usage

### Primary Logo
- Use on white backgrounds
- Minimum size: 100px width
- Clear space: 50% of logo height

### Logo Variations
- Full color (primary)
- Monochrome black
- Monochrome white
- Icon only (for small spaces)

**Don't:**
- Stretch or distort
- Change colors
- Add effects
- Use on busy backgrounds

## Color Palette

### Primary Colors
- **Acme Blue**: #0066CC
- **Acme Gray**: #333333
- **White**: #FFFFFF

### Secondary Colors
- **Success Green**: #00CC66
- **Warning Orange**: #FF9933
- **Error Red**: #CC0033

### Usage Guidelines
- Primary: Main CTAs, headers
- Secondary: Status indicators, accents
- Maintain 4.5:1 contrast ratio for accessibility

## Typography

### Headings
- Font: Inter Bold
- Sizes: 48px (H1), 36px (H2), 24px (H3)
- Line height: 1.2

### Body Text
- Font: Inter Regular
- Size: 16px
- Line height: 1.6

### Code
- Font: JetBrains Mono
- Size: 14px

## Voice & Tone

### Voice (consistent)
- Professional yet approachable
- Clear and concise
- Confident but not arrogant

### Tone (varies by context)
- **Marketing**: Enthusiastic, inspiring
- **Documentation**: Helpful, precise
- **Support**: Empathetic, solution-focused
- **Error messages**: Calm, actionable

## Writing Style

- Use active voice
- Write in second person ("you")
- Keep sentences under 20 words
- Avoid jargon and buzzwords
- Use contractions (we're, don't)

## Examples

**Good**: "You'll love how easy it is to get started."
**Bad**: "Users will find that the onboarding process has been optimized."

**Good**: "Sorry, something went wrong. Try refreshing the page."
**Bad**: "Error 500: Internal Server Exception."

## Image Guidelines

- Use authentic photos (not stock)
- Diverse representation
- Landscape orientation for hero images
- Optimize for web (< 500KB)

## Social Media

### Profile Images
- Logo icon on brand blue background
- 400x400px minimum

### Cover Images
- Brand pattern or product screenshot
- 1200x630px

### Post Guidelines
- Include relevant hashtags (max 3)
- Tag mentioned accounts
- Use consistent filters
